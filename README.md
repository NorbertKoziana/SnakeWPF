# Snake Game in C# (WPF)

A simple implementation of the Snake game built using **C#** and **WPF**.

---

## How to Play
1. Use arrow keys to control the direction of the snake.
2. Eat food to grow longer and increase your score.
3. Avoid hitting yourself!
